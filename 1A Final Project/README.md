Nautical knowledge quiz game that saves user high-score locally.

2 quizzes 'Novice' & 'Advanced'.
Each quiz has a database of 20 questions with 10 of them selected at random for each quiz attempt.

'question' class
'multiple choice' sub-sclass
'select all that apply' sub-class

Keep track of questions answered correctly & incorrectly.
Timer that shows how long it took to take the quiz.
End of quiz screen that displays score & time compared to best score & time.

Best score if first chosen off of % questions correct, 
then if there are multiple attempts with the same % correct,
highest time out of those scores is the 'high score'.

Example Top-Scores:
1. Score: 7/10 Time: 4:00
2. Score: 7/10 Time: 4:32
3. Score 6/10 Time: 3:22
